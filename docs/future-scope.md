# Future Scope Enhancements

Items collected during Phase 1-3 development. To be planned in Phase 5+.

## Multi-Persona & Teams
- **Multiple personas** — each with their own calendar links for availability
- **Teams/departments** (sales, finance, logistics) — route calls based on topic and availability
- Multiple user profiles with independent availability rules

## Calendar Integration
- **Google Calendar** as an alternative/additional calendar source (alongside MS Graph)
- Calendar-based routing rules

## Call Control
- **Call blocking** — block by full number, number sequence, or region/country code
- Blocked calls answered with "your call cannot be completed as dialed" or similar
- Caller ID lookup (reverse phone number search)

## Notifications & Channels
- Push notifications via Pushover/Ntfy
- SMS/WhatsApp messaging integration

## Analytics & History
- Call recording audio playback in dashboard
- Analytics dashboard (calls/week, avg duration, common topics)
- Call export to CSV

## Other
- Mobile-responsive dashboard
- Swiss German / multilingual support
- Multi-tenant support
